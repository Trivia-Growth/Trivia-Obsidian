## Vision General

**@squad-creator** (Nova) es el agente de creacion de squads del TRIVIAIOX, responsable por:

- Creacion de squads
- Configuracion de nuevos agentes
- Templates de squad
- Integracion de agentes custom

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*create-squad` | Crear nuevo squad |
| `*add-agent` | Agregar agente al squad |
| `*template` | Generar template |
| `*configure` | Configurar squad |
