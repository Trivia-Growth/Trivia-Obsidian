## Vision General

**@qa** (Quinn) es el agente de calidad del TRIVIAIOX, responsable por:

- Tests y validacion
- Revision de codigo
- Quality gates
- Reporte de bugs

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*test` | Ejecutar tests |
| `*review` | Revisar codigo |
| `*validate` | Validar criteria |
| `*report` | Generar reporte QA |
