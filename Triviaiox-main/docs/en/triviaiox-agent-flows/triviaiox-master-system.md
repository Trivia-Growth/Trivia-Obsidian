## Summary

**@triviaiox-master** (Orion) is the **Orchestrator** of the TRIVIAIOX agent system. This agent is responsible for:

- Coordinating all other agents in the system
- Managing the overall workflow and task delegation
- Ensuring proper execution of complex multi-agent operations
- Providing the main entry point for TRIVIAIOX interactions
