# License Clarification (Core vs Pro)

**Last updated:** 2026-02-15

This document clarifies the licensing boundary in TriviaGrowth Triviaiox.

## Scope

- `triviaiox-core` (this repository): **MIT**
- `triviaiox-pro` (private/proprietary module): separate license terms defined in that module/repository

## What This Means

- The open-source core framework is MIT-licensed and can be used under MIT terms.
- Commercial or proprietary restrictions, if any, belong to `triviaiox-pro` and do not change the MIT license of `triviaiox-core`.

## Practical Guidance

- If you are using only `triviaiox-core`, follow MIT terms in `LICENSE`.
- If you are using any `triviaiox-pro` capability, also review and comply with `triviaiox-pro` license terms.

## Related Files

- `LICENSE` (root): MIT license for `triviaiox-core`
- `docs/legal/terms.md`: terms of use for this repository
