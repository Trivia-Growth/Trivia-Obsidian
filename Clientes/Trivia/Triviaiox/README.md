## O que é o Triviaiox?

Sistema de orquestração de IA para desenvolvimento full stack. Permite ativar **agentes especializados** (dev, qa, pm, etc.) diretamente no Claude Code, Cursor, Gemini CLI ou Codex CLI, cada um com persona, comandos e memória próprios.

**Premissa central:** CLI First → Observability Second → UI Third.
