## Como foi exportado (caminho no Literarius)

```
Literarius → Utilitários → Exportar → Produtos

Configurações de exportação:
  Situação de Estoque: TODOS INDEPENDENTE DE SALDO
  Status:              TODOS OS PRODUTOS
  Filtro:              Funil ao lado de "Buscar" → selecionar Editora

Obs. (Lucas Mococa):
  - Na frente do campo "Código" é possível adicionar/remover colunas antes de buscar
  - No filtro ao lado de "Buscar" é possível selecionar editora
  - Nossa editora tem livros como "ASSOCIAÇÃO EDITORA PRESBITERIANA DE PINHEIROS"
    e também como "HEZIOM"
  - Após buscar, clicar em "Exportar" para gerar o XLSX

Script usado na exportação: SQL00688.sql (visível na barra de status do Literarius)
```

## Correlação com o HeziomOS

A planilha confirma os campos disponíveis para o sync Literarius → Tray:

| Campo na Planilha | Campo Literarius | Campo Tray | Usado em |
|---|---|---|---|
| Código | `Produto.CodProduto` | — | Chave interna |
| Título | `Produto.Titulo` | `product.name` | Fase 2.2 Catálogo |
| ISBN | `Produto.ISBN` | `product.reference` | Conciliação |
| Preço | `Produto.PrecoVenda` | `product.price` | Sync de preço |
| Custo | `Produto.CustoMedio` | — | CMV / margem |
| Saldo Estoque | `vwProdutoEstoque.QtdeEstoque` | `product.stock` | **Fase 1.1 (prioridade)** |
| Reservado | `vwProdutoEstoque.QtdeReservado` | — | Controle interno |
| Disponível | `vwProdutoEstoque.QtdeDisponivel` | `product.stock` | **Fase 1.1** |
| Última Venda | `Produto.UltimaVenda` | — | Giro / reposição |
| Editora Fantasia | `Produto.EditoraFantasia` | — | Filtro no Literarius |

> Para o sync de estoque (Fase 1.1), o campo correto é **Disponível** (saldo − reservado), não o Saldo bruto.
