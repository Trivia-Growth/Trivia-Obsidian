# Tray — Temas e Marketplace de Parceiros

> O marketplace de temas da Tray **existe e é aberto** — qualquer parceiro certificado pode publicar e vender temas.
> URL: `https://temas.tray.com.br` · 819+ temas disponíveis

---

## O marketplace de temas (temas.tray.com.br)

### Escala e preços

| Faixa | Exemplos de preço |
|---|---|
| Gratuitos | — |
| Até R$ 599 | Entrada |
| Até R$ 1.799 | Mid-market |
| A partir de R$ 1.800 | Premium |

- **Pagamento único** — sem mensalidade ou anuidade após a compra
- Após confirmação do pagamento, tema fica imediatamente disponível para instalação

### Categorias existentes

Moda e Acessórios · Cosméticos/Saúde/Beleza · Eletrônicos · Casa/Decoração · Peças Automotivas · Joias · Esporte/Fitness · Petshop · Infantil · Ótica · Informática

> ⚠️ **Não existe categoria de Livros, Editoras ou Livrarias** — gap de mercado confirmado.

## Programa de Parceiros — como funciona

### 4 pilares de geração de receita

| Pilar | Modelo | Receita |
|---|---|---|
| **Loja de Temas** | Criar e vender temas no marketplace | Venda direta (% da Tray a confirmar) |
| **Referência de Lojas** | Indicar clientes para a Tray | **20% recorrente** sobre mensalidade do cliente indicado |
| **Loja de Apps** | Desenvolver apps/integrações | Receita única ou recorrente |
| **Implementação** | Serviços de implantação de loja | Fluxo de caixa imediato |

### Níveis de certificação

`Starter → Bronze → Silver → Gold → Diamond`

Baseado em engajamento no ecossistema e performance financeira.

### O que o parceiro recebe

- Acesso a **APIs públicas** da Tray
- Acesso ao **OpenCode** (desenvolvimento de temas em HTML/CSS/JS)
- **Loja de desenvolvimento/teste gratuita**
- Selo de parceiro certificado (visível no marketplace)
- Suporte direto com o time de parcerias

### Tecnologia para desenvolvimento de temas

- **OpenCode** — editor HTML/CSS/JS nativo da Tray
- Sem CLI externo confirmado
- Sem template engine documentado publicamente (a confirmar com parceiros Tray)

## 🔴 Oportunidade estratégica para a Trivia

### Gap identificado: zero temas para editoras/livrarias

Das 819 temas no marketplace, **nenhum** é voltado para:
- Editoras (livros físicos, e-books, kits)
- Livrarias
- Conteúdo cristão/religioso
- Produtos editoriais com múltiplos autores

A Heziom seria o **caso de uso perfeito** para construir e validar esse tema.

### Proposta de valor

| Ação | O que gera |
|---|---|
| Trivia vira parceira Tray | Acesso ao OpenCode + loja teste + suporte |
| Desenvolve tema editorial para Heziom | Case real, cliente pagador |
| Publica no marketplace | Primeiro (e único) tema para editoras na Tray |
| Outras editoras/livrarias compram | Receita passiva para a Trivia |

### Potencial de mercado

- Tray tem **centenas de editoras e livrarias** como clientes (mercado editorial usa muito Tray)
- Tema otimizado para livros: ISBN, autores, sinopse, séries, kits, e-books
- Preço estimado de mercado: R$ 1.200–R$ 2.500 (faixa mid-market)
- Se 50 editoras comprarem: **R$ 60k–R$ 125k** em receita de tema

## Diferença: Tema vs App de Integração

| | Tema (front-end) | App de Integração (back-end) |
|---|---|---|
| **O que é** | Layout visual da loja | Lógica + dados |
| **Tecnologia** | HTML/CSS/JS via OpenCode | REST API + OAuth |
| **Marketplace** | temas.tray.com.br ✅ | Após homologação ✅ |
| **O HeziomOS é** | ❌ | ✅ |
| **São independentes?** | Sim — podem evoluir separado |  |

A Trivia pode entregar **os dois** para a Heziom:
1. **Tema editorial** → loja bonita e otimizada para livros
2. **HeziomOS (app)** → dashboard CEO + sync Literarius + automações
