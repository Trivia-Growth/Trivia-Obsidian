# Pacote para Claude Design — LP Bíblia 120 Anos (IPP)

Tudo o que você precisa para desenvolver o layout da landing page no Claude.

## O que tem aqui

| Pasta / arquivo | Para que serve |
|---|---|
| **`BRIEFING-DESIGN.md`** | Briefing completo: identidade visual, paleta, fontes, estrutura, conversão e lista de ativos. **Cole no Claude ao começar.** |
| **`TEXTOS-PRONTOS.md`** | Toda a copy pronta (hero, sinopse, história, citações, FAQ, CTAs). |
| **`imagens/`** | Imagens otimizadas (7 MB) prontas para subir no Claude: `marca/`, `produto/`, `templo/`, `historia/`. |
| **`referencia-lp/`** | Versão funcional já construída desta LP. Abra o `index.html` no navegador para ver uma realização do briefing antes de iterar. |

## Como usar no Claude

1. Abra um chat novo no Claude (claude.ai).
2. Cole o conteúdo de **`BRIEFING-DESIGN.md`** e de **`TEXTOS-PRONTOS.md`**.
3. Suba as imagens da pasta **`imagens/`** que quiser usar (logos, mockups, fotos).
4. Peça o layout (ex: "monte essa landing page como um artifact, mescla claro/escuro, seguindo o briefing").
5. Itere o visual conversando com o Claude.

> Dica: para o Claude ver o ponto de partida, abra `referencia-lp/index.html` e descreva o que quer mudar, ou tire prints e suba junto.

## Lembretes de marca

- Mescla **claro + escuro** (escuro = couro premium dourado; claro = história editorial).
- Símbolo da **tulipa** como motivo recorrente.
- Copy: **nunca** usar travessão "—" nem a palavra "jornada".
