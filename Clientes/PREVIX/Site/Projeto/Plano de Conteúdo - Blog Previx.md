## Especificações Técnicas (Jimmy 3.0)

Cada artigo DEVE conter:

- **Lede:** 40-60 palavras, pirâmide invertida (conclusão primeiro)
- **Conclusões Principais:** 3-5 fatos atômicos, 40-280 caracteres cada
- **Estatísticas:** Mínimo 3, cada com fonte + URL verificável
- **FAQ:** 4-8 perguntas, respostas de 40-150 palavras (gera FAQPage Schema)
- **Corpo MDX:** H2/H3 como perguntas, blocos de 50-150 palavras
- **Imagem de capa:** Foto profissional (usar stock ou acervo Previx)
- **Meta description:** Até 180 chars, keyword principal + CTA implícito
- **Schema JSON-LD:** Article, BlogPosting ou HowTo conforme indicado
